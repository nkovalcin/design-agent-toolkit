# ČASŤ 1: COPYWRITING FOUNDATIONS

## Marketing Philosophy

### Prečo väčšina marketingu nefunguje

```
PROBLÉM #1: Feature-focused
"Náš produkt má AI-powered analytics s real-time dashboardom"
→ Zákazníka nezaujíma AKO, zaujíma ho ČO TO PRE NEHO ZNAMENÁ

PROBLÉM #2: Company-focused
"Sme lídri v oblasti digitálnych riešení s 15-ročnou históriou"
→ Zákazníka nezaujímajú TVOJE úspechy, zaujímajú ho JEHO problémy

PROBLÉM #3: Vague claims
"Najlepšie riešenie na trhu"
→ Bez dôkazu sú to len prázdne slová

PROBLÉM #4: Too clever
"Synergizujte váš workflow pre optimálnu produktivitu"
→ Nikto nevie čo to znamená
```

### Základné princípy

```
1. CLARITY > CLEVERNESS
   Jasný text vždy porazí "kreatívny" text

2. BENEFITS > FEATURES
   Feature: "256-bit šifrovanie"
   Benefit: "Vaše dáta sú v bezpečí"

3. SPECIFIC > GENERIC
   Generic: "Ušetríte čas"
   Specific: "Ušetríte 5 hodín týždenne"

4. CUSTOMER > COMPANY
   Company: "Sme najlepší poskytovateľ..."
   Customer: "Získate..."

5. CONVERSATION > PRESENTATION
   Píš ako hovoríš, nie ako píšeš esej
```

### The "So What?" Test

Po každej vete sa spýtaj "So what?" (A čo má byť?):

```
"Náš softvér používa AI."
So what?
"Takže automaticky kategorizuje vaše emaily."
So what?
"Takže strávite o 2 hodiny menej denne v inboxe."
So what?
"Takže máte viac času na prácu, ktorá vás baví."

→ TOTO je skutočný benefit
```

## 1. Customer Research

### Kto je tvoj zákazník?

```
DEMOGRAFICKÉ:
- Vek, pohlavie, lokalita
- Príjem, vzdelanie
- Profesia, pozícia

PSYCHOGRAFICKÉ:
- Hodnoty a presvedčenia
- Záujmy a koníčky
- Životný štýl
- Osobnostné črty

SITUAČNÉ:
- Aktuálna situácia
- Čo spustilo hľadanie riešenia?
- Aké riešenia už vyskúšali?
- Prečo nefungovali?
```

### Customer Research Methods

```
1. INTERVIEWS
   - 5-10 rozhovorov s existujúcimi zákazníkmi
   - "Čo vás priviedlo k nám?"
   - "Čo ste robili predtým?"
   - "Čo by vám najviac chýbalo?"

2. SURVEYS
   - NPS (Net Promoter Score)
   - Satisfaction surveys
   - Exit surveys

3. REVIEWS MINING
   - Vlastné reviews
   - Reviews konkurencie
   - Amazon, G2, Capterra, Trustpilot

4. SUPPORT TICKETS
   - Najčastejšie otázky
   - Najčastejšie problémy
   - Jazyk, ktorý zákazníci používajú

5. SOCIAL LISTENING
   - Reddit, Twitter, Facebook groups
   - Fóra v tvojom niche
   - Komentáre pod článkami
```

### Voice of Customer (VoC) Database

Zbieraj presné frázy, ktoré zákazníci používajú:

```
BOLESTI (Pain):
"Strácam hodiny denne manuálnym zadávaním dát"
"Neviem kde sú moje súbory"
"Môj šéf ma núti používať Excel a je to peklo"

TÚŽBY (Desires):
"Chcem mať všetko na jednom mieste"
"Potrebujem to zvládnuť aj bez IT oddelenia"
"Chcem konečne odísť z práce načas"

PREKÁŽKY (Objections):
"Vyzerá to komplikovane"
"Nemám čas sa učiť nový nástroj"
"Čo ak to nebude fungovať s našimi dátami?"

TRIGGER EVENTS:
"Keď sme stratili dôležitý súbor..."
"Keď som strávil celý víkend v práci..."
"Keď konkurencia získala nášho klienta..."
```

```

DEMOGRAFICKÉ:

- Bratislava
- Marketing manažér, 5 rokov skúseností
- Príjem: 2500€/mesiac

NÁMIETKY:
```

### Value Proposition Formula

```
PRE [cieľový zákazník]
KTORÝ [má tento problém/potrebu]
NÁŠ PRODUKT JE [kategória produktu]
KTORÝ [kľúčový benefit]
NA ROZDIEL OD [konkurencia/alternatíva]
MY [kľúčová diferenciácia]

PRÍKLAD:
Pre marketingových manažérov,
ktorí trávia hodiny manuálnym reportovaním,
Analytika Pro je all-in-one reporting platforma,
ktorá automaticky generuje týždenné reporty,
na rozdiel od Google Analytics a Excel kombinácie,
my integrujeme všetky vaše dátové zdroje na jedno miesto.
```

### Unique Value Proposition (UVP) Formats

**Format 1: Headline + Subheadline**
```
Headline: Reporty za 5 minút, nie 5 hodín.
Subheadline: Automatické marketingové reporty z všetkých vašich nástrojov.
```

**Format 2: [End Result] + [Time Period] + [Addressing Objection]**
```
Zdvojnásobte konverzie za 30 dní bez najímania agentúry.
```

**Format 3: [Action] + [Benefit] + [Specific Result]**
```
Automatizujte svoj marketing a ušetrite 10+ hodín týždenne.
```

**Format 4: We help [X] do [Y] by [Z]**
```
Pomáhame e-shopom zvýšiť tržby optimalizáciou checkout procesu.
```

## 3. Voice & Tone

### Brand Voice Attributes

```
PRÍKLAD: Slack

VOICE ATTRIBUTES:
1. Clear (nie corporate)
2. Concise (nie lengthy)
3. Human (nie robotic)
4. Playful (nie boring)

VOCABULARY:
✓ "Daj vedieť tímu"
✗ "Notifikujte stakeholderov"

✓ "Ups, niečo sa pokazilo"
✗ "Error 500: Internal Server Error"

✓ "Poďme na to!"
✗ "Pokračujte v procese"
```

### Voice Matrix

```
              CASUAL ←──────────────→ FORMAL
                    │                │
PLAYFUL     ────────┼────────────────┼──────── SERIOUS
                    │    ★ YOUR     │
IRREVERENT  ────────┼─── BRAND ─────┼──────── RESPECTFUL
                    │                │
ENTHUSIASTIC────────┼────────────────┼──────── MATTER-OF-FACT
```

### Tone Shifts by Context

```
VOICE = Konzistentná osobnosť značky
TONE = Prispôsobenie kontextu

SITUÁCIA          | TONE SHIFT
------------------|------------------
Homepage          | Confident, inspiring
Error message     | Helpful, reassuring
Success message   | Celebratory
Pricing page      | Clear, transparent
Support docs      | Patient, thorough
Social media      | Casual, engaging
Legal pages       | Formal, precise
```

### Writing Guidelines

```
DO:
✓ Použi "vy" nie "používatelia"
✓ Použi aktívny hlas
✓ Použi krátke vety
✓ Použi konkrétne čísla
✓ Použi slová, ktoré používa zákazník

DON'T:
✗ Nepouži žargón bez vysvetlenia
✗ Nepouži pasívny hlas
✗ Nepouži dlhé odseky
✗ Nepouži vágne tvrdenia
✗ Nepouži superlatives bez dôkazu
```

## 4. AIDA Framework

### Attention → Interest → Desire → Action

```
ATTENTION (Pozornosť)
Upútaj pozornosť v prvej sekunde.
- Šokujúce štatistiky
- Provokujúce otázky
- Bold statements
- Neočakávané obrazy

INTEREST (Záujem)
Udrž pozornosť relevanciou.
- Identifikuj ich problém
- Ukáž že rozumieš
- Predstav možnosti
- Buduj zvedavosť

DESIRE (Túžba)
Premeň záujem na túžbu.
- Benefity, nie features
- Social proof
- Vizualizácia výsledku
- Emocionálna spojitosť

ACTION (Akcia)
Povedz presne čo robiť.
- Jasné CTA
- Znížená frikccia
- Urgentnosť
- Garancie
```

### AIDA Príklad: Landing Page

```
ATTENTION:
"97% e-shopov stráca zákazníkov kvôli pomalému webu."

INTEREST:
"Viete, že každá sekunda načítavania stojí 7% konverzií?
Analyzovali sme 10,000 e-shopov a zistili sme,
že priemerný web stráca €47,000 ročne len kvôli rýchlosti."

DESIRE:
"SpeedBoost zrýchli váš web o 300% za 24 hodín.
Naši klienti zaznamenali priemerne 23% nárast konverzií.

'Konečne som prestala strácať zákazníkov. Tržby išli hore
o 35% za prvý mesiac.' — Jana K., majiteľka e-shopu"

ACTION:
"Získajte bezplatnú analýzu rýchlosti vášho webu.
[Analyzovať môj web zadarmo]
Žiadna kreditná karta. Výsledky za 60 sekúnd."
```

## 5. PAS Framework

### Problem → Agitate → Solution

```
PROBLEM (Problém)
Pomenuj problém, ktorý cítia.
- Buď konkrétny
- Použi ich jazyk
- Ukáž že rozumieš

AGITATE (Rozvír)
Zosilni bolesť problému.
- Dôsledky ignorovania
- Čo to stojí (čas, peniaze, stres)
- Emocionálny dopad
- Budúce problémy

SOLUTION (Riešenie)
Predstav riešenie ako úľavu.
- Ako to rieši problém
- Prečo práve toto riešenie
- Dôkazy že to funguje
```

### PAS Príklad: Email

```
PROBLEM:
Trávite každý pondelok ráno 3 hodiny kopírovaním
dát z rôznych nástrojov do Excelu?

AGITATE:
Kým vy manuálne zbieraťe čísla, vaša konkurencia
už reaguje na trendy.

Tie 3 hodiny týždenne = 156 hodín ročne.
To je takmer mesiac práce stratenej na copy-paste.

A čo horšie - manuálne dáta obsahujú chyby.
Jeden preklik a váš report ukazuje úplne zlé čísla.
Rozhodnutia založené na zlých dátach = zlé rozhodnutia.

SOLUTION:
ReportBot automaticky sťahuje dáta z 50+ nástrojov
a generuje report za 5 minút.

Žiadne kopírovanie. Žiadne chyby. Žiadne pondelkové rána v Exceli.

[Vyskúšajte zadarmo na 14 dní]
```

## 6. BAB Framework

### Before → After → Bridge

```
BEFORE (Pred)
Opíš súčasnú situáciu zákazníka.
- Ich realita teraz
- Problémy a frustrácie
- Čo im chýba

AFTER (Po)
Namaľuj obraz lepšej budúcnosti.
- Život po vyriešení problému
- Konkrétne zlepšenia
- Emocionálny stav

BRIDGE (Most)
Ukáž ako sa dostať z Before do After.
- Tvoj produkt/služba
- Ako to funguje
- Prečo je to jednoduché
```

### BAB Príklad: Homepage Section

```
BEFORE:
Každý večer sedíte nad faktúrami do polnoci.
Excel tabuľky, papierové bločky, stratené účtenky.
Účtovníčka vám volá, že niečo nesedí.
Dane sú stres, nie rutina.

AFTER:
Predstavte si, že všetky výdavky sú automaticky
zaznamenané a kategorizované.
Že účtovníčka dostane všetko pripravené jedným klikom.
Že dane podáte za hodinu, nie za týždeň.
Že večery trávite s rodinou, nie s faktúrami.

BRIDGE:
Fakturoid automatizuje celé vaše účtovníctvo.
Naskenujte účtenku telefónom → automaticky sa zaradí.
Vystavte faktúru za 30 sekúnd → automaticky sa odošle.
Exportujte pre účtovníka → všetko v správnom formáte.

[Začnite zadarmo]
```

## 7. 4Ps Framework

### Promise → Picture → Proof → Push

```
PROMISE (Sľub)
Hlavný benefit, ktorý ponúkaš.
- Jasný, konkrétny výsledok
- Čo zákazník získa

PICTURE (Obraz)
Vizualizácia výsledku.
- Ako bude vyzerať život
- Senzorické detaily
- Emocionálna projekcia

PROOF (Dôkaz)
Prečo ti môžu veriť.
- Testimonials
- Case studies
- Štatistiky
- Credentials

PUSH (Postrčenie)
Výzva k akcii.
- Čo presne spraviť
- Prečo teraz
- Odstránenie rizika
```

### 4Ps Príklad: Sales Page Section

```
PROMISE:
Naučíme vás hovoriť po anglicky plynule za 90 dní.
Nie teóriu. Nie gramatiku. Skutočnú konverzáciu.

PICTURE:
Predstavte si, že na ďalšej pracovnej ceste
s istotou vediete meeting v angličtine.
Že na dovolenke flirtujete s barmanom v Miami.
Že pozeráte Netflix bez titulkov a rozumiete vtipom.
Že na LinkedIn sebavedomne píšete zahraničným recruiterom.

PROOF:
"Za 3 mesiace som prešla z A2 na B2 a dostala prácu
v medzinárodnej firme." — Katka, 28

4,847 absolventov
4.8/5 priemerné hodnotenie
89% dokončí kurz (priemer branže je 15%)

PUSH:
Najbližší kurz začína o 7 dní.
Zostáva 12 miest z 30.

Ak po 30 dňoch nevidíte pokrok, vrátíme vám 100% peňazí.
Žiadne otázky.

[Rezervovať miesto za €297]
```

## 8. ACCA Framework

### Awareness → Comprehension → Conviction → Action

```
AWARENESS (Povedomie)
Uvedom zákazníka o probléme/príležitosti.
- Nové informácie
- Prekvapivé fakty
- "Nevedeli ste že..."

COMPREHENSION (Pochopenie)
Pomôž pochopiť problém a riešenie.
- Vysvetli príčiny
- Edukuj o riešení
- Odstráň zmätok

CONVICTION (Presvedčenie)
Presvedč že tvoje riešenie je správne.
- Dôkazy
- Porovnania
- Logické argumenty
- Emocionálne argumenty

ACTION (Akcia)
Motivuj k akcii.
- Jasné CTA
- Odstránenie bariér
- Urgentnosť
```

### ACCA Príklad: Blog to Conversion

```
AWARENESS:
Vedeli ste, že 68% nákupných košíkov je opustených?
To znamená, že 2 z 3 ľudí, ktorí chcú kúpiť,
nakoniec nekúpia.

COMPREHENSION:
Najčastejšie dôvody:
1. Nečakané náklady na dopravu (48%)
2. Príliš komplikovaný checkout (24%)
3. Požadovaná registrácia (18%)

Problém nie je váš produkt.
Problém je proces nákupu.

CONVICTION:
E-shopy, ktoré zjednodušili checkout,
zaznamenali priemerne 35% nárast konverzií.

Náš klient Modastyl.sk znížil opustené košíky
z 72% na 41% za 2 týždne.

"Investícia sa vrátila za 11 dní." — CEO Modastyl

CONVICTION:
Bezplatný Checkout Audit odhalí presne,
kde strácate zákazníkov.

[Získať bezplatný audit]
Odpoveď do 24 hodín.
```

## 9. StoryBrand Framework

### Donald Miller's 7-Part Framework

```
1. CHARACTER (Hrdina)
   Zákazník je hrdina, nie vy.
   → Čo chce?

2. PROBLEM (Problém)
   External, Internal, Philosophical
   → Čo mu stojí v ceste?

3. GUIDE (Sprievodca)
   Vy ste sprievodca (Yoda, nie Luke)
   → Empatia + Autorita

4. PLAN (Plán)
   Jednoduchý plán na úspech
   → 3 kroky maximum

5. CALL TO ACTION
   Priamy a prechodný CTA
   → Čo presne má spraviť?

6. SUCCESS (Úspech)
   Ako vyzerá úspech?
   → Pozitívny výsledok

7. FAILURE (Zlyhanie)
   Čo ak neurobí nič?
   → Negatívny výsledok (stakes)
```

### StoryBrand BrandScript Template

```
CHARACTER:
[Zákazník] chce [túžba].

PROBLEM:
Externý: [praktický problém]
Interný: [ako sa cíti]
Filozofický: [prečo je to nesprávne]

GUIDE:
Rozumieme [empatia].
Pomohli sme [credentials].

PLAN:
1. [Krok 1]
2. [Krok 2]
3. [Krok 3]

CTA:
[Primárne CTA]
[Sekundárne CTA]

SUCCESS:
[Pozitívny výsledok]

FAILURE:
[Negatívny výsledok ak nič neurobí]
```

### StoryBrand Príklad: Fitness App

```
CHARACTER:
Zaneprázdnení profesionáli chcú byť fit a zdraví.

PROBLEM:
Externý: Nemajú čas chodiť do posilňovne.
Interný: Cítia sa vinní, že zanedbávajú zdravie.
Filozofický: Nemali by si musieť vyberať medzi
             kariérou a zdravím.

GUIDE:
Empathy: Vieme aké je to mať 12-hodinové dni
         a nulovú energiu na cvičenie.
Authority: Vytvorili sme programy pre 50,000 ľudí
           s priemerným úbytkom 8kg za 90 dní.

PLAN:
1. Stiahnite si app (2 minúty)
2. Vyberte 15-minútový program
3. Cvičte doma, kedykoľvek

CTA:
Primárne: "Začať 7-dňovú skúšku zadarmo"
Sekundárne: "Pozrieť ukážkový tréning"

SUCCESS:
Budete mať energiu na celý deň,
zhodíte nadbytočné kilá,
a konečne sa budete cítiť dobre vo vlastnom tele.

FAILURE:
Bez zmeny budete naďalej unavení,
frustrovaní svojím telom,
a riskujete zdravotné problémy v budúcnosti.
```

## 10. Jobs To Be Done

### JTBD Framework

```
CORE CONCEPT:
Ľudia nekupujú produkty.
Kupujú "pokrok" v konkrétnej situácii.

FORMULA:
Keď [situácia],
chcem [motivácia/cieľ],
aby som [očakávaný výsledok].

PRÍKLAD:
Keď čakám na letisko,
chcem zabiť čas,
```

### Real vs Fake Urgency

```

- Limited inventory (actually limited)
- Time-limited offer (actually expires)
- Event date (actual deadline)

FAKE (UNETHICAL):

- "Only 3 left" that never changes
- "Offer ends tonight" every night
- Artificial scarcity

→ Fake urgency destroys trust when discovered
```


# ČASŤ 5: PSYCHOLOGY & PERSUASION

## 23. Cognitive Biases

### Key Biases for Marketing

```
ANCHORING:
First number influences perception.
→ Ukáž pôvodnú cenu pred zľavou
→ Ukáž najdrahší plán prvý

SOCIAL PROOF:
People follow others.
→ "50,000+ customers"
→ "Most popular" badge

SCARCITY:
Rare = Valuable
→ "Only 5 left"
→ "Limited time offer"

LOSS AVERSION:
Losing hurts 2x more than gaining feels good.
→ "Don't miss out"
→ "Stop losing customers"

RECIPROCITY:
We want to return favors.
→ Give value first (lead magnet)
→ Free trial, free consultation

AUTHORITY:
We trust experts.
→ Expert endorsements
→ Certifications, credentials

MERE EXPOSURE:
Familiarity breeds preference.
→ Retargeting ads
→ Email nurturing
→ Consistent branding

SUNK COST:
We continue what we started.
→ Progress bars
→ "You're almost there!"
→ Free trials that require setup

PEAK-END RULE:
We remember peaks and endings.
→ Memorable onboarding
→ Great customer experience at checkout
→ Follow-up after purchase
```

## 25. Persuasion Principles

### Cialdini's 7 Principles

```
1. RECIPROCITY
   Give before you ask.
   → Free content, tools, trials

2. COMMITMENT & CONSISTENCY
   Start with small asks.
   → Micro-commitments, quizzes
   → "Say yes" progression

3. SOCIAL PROOF
   People follow people.
   → Testimonials, numbers, activity

4. AUTHORITY
   Expert opinions matter.
   → Credentials, media, endorsements

5. LIKING
   We buy from people we like.
   → Personality, relatability, story

6. SCARCITY
   Less = More valuable.
   → Limited time/quantity

7. UNITY
   Shared identity creates bonds.
   → "Fellow marketers..."
   → "As a parent myself..."
```

### Fogg Behavior Model

```
BEHAVIOR = Motivation × Ability × Trigger

MOTIVATION:
- Pain/Pleasure
- Hope/Fear
- Acceptance/Rejection

ABILITY (reduce friction):
- Time
- Money
- Physical effort
- Mental effort
- Social deviance
- Non-routine

TRIGGER (prompt at right time):
- Spark (low motivation) — inspire
- Facilitator (low ability) — make easy
- Signal (high both) — just remind
```

### PASTOR Framework

```
P — Problem
    Identify the pain

A — Amplify
    Agitate the consequences

S — Story/Solution
    Tell story, present solution

T — Transformation
    Show the change

O — Offer
    Make the offer

R — Response
    Call to action

PRÍKLAD:
P: "Tired of losing customers at checkout?"
A: "Every abandoned cart is money left on the table.
    At 68% average abandonment, you could be losing
    thousands every month."
S: "That's why we built CartSaver. When John from
    TechStore implemented it, he recovered €15,000
    in the first month."
T: "Imagine converting even 10% more of those
    abandoned carts. That's [their potential €]."
O: "CartSaver starts at €49/month with a 14-day
    free trial."
R: "Start your free trial now — setup takes 5 minutes."
```

# ČASŤ 6: OPTIMIZATION

## 26. A/B Testing Copy

### What to Test (Priority Order)

```
HIGH IMPACT:
1. Headlines
2. CTAs (text and design)
3. Value proposition
4. Social proof placement
5. Pricing presentation

MEDIUM IMPACT:
6. Subheadlines
7. Body copy length
8. Feature ordering
9. Email subject lines
10. Form fields

LOW IMPACT:
11. Button colors
12. Minor wording changes
13. Image variations
14. Font changes
```

### A/B Test Framework

```
1. HYPOTHESIS
   "Ak zmeníme [X], potom [Y] sa zlepší,
   pretože [dôvod]."

2. METRICS
   Primary: Conversion rate
   Secondary: Bounce rate, time on page

3. SAMPLE SIZE
   Calculate required size before starting
   (tools: Optimizely, VWO calculators)

4. DURATION
   At least 1-2 business cycles
   Minimum 100 conversions per variant

5. ANALYSIS
   Statistical significance > 95%
   Consider practical significance too
```

### Copy Test Examples

```
HEADLINE TEST:
Control: "Project Management Made Simple"
Variant A: "Get More Done in Less Time"
Variant B: "The Last PM Tool You'll Ever Need"
Variant C: "Join 50,000+ Teams Who Ship Faster"

CTA TEST:
Control: "Start Free Trial"
Variant A: "Try Free for 14 Days"
Variant B: "Start Building Today"
Variant C: "Get Started Free — No Card Required"

PRICING TEST:
Control: "€99/month"
Variant A: "€99/month (Save €500/year)"
Variant B: "Less than €4/day"
Variant C: "€99/month — 30-day money back"
```

## 27. Conversion Audit

### Page Audit Checklist

```
CLARITY (5 seconds test):
□ Do I know what this company does?
□ Do I know who it's for?
□ Do I know what to do next?

VALUE PROPOSITION:
□ Is benefit clear?
□ Is it specific (not generic)?
□ Is it differentiated from competition?

HEADLINES:
□ Is main headline compelling?
□ Does it address customer, not company?
□ Is there clear hierarchy?

SOCIAL PROOF:
□ Are there testimonials?
□ Are they specific with results?
□ Are they credible (photos, names)?
□ Are there trust badges?

CTA:
□ Is CTA visible above fold?
□ Is CTA text action-oriented?
□ Is there benefit in CTA?
□ Are CTAs placed throughout page?

OBJECTION HANDLING:
□ Is there FAQ?
□ Is there guarantee?
□ Are common concerns addressed?

URGENCY/SCARCITY:
□ Is there reason to act now?
□ Is it authentic?

FRICTION:
□ Are forms minimal?

□ Are there unnecessary steps?
```
