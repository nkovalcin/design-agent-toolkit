# Project Kickoff Cheatsheet

> Prejdi tento dokument PRED začiatkom každého projektu.
> Zodpovedaj všetky otázky. Urob rozhodnutia. Až potom začni.
> Verzia 1.0

---

## Ako používať tento dokument

```
1. Vytlač alebo skopíruj do nového súboru
2. Prejdi každú sekciu
3. Vyplň odpovede / zaškrtni rozhodnutia
4. Ak nevieš odpovedať → zisti pred začiatkom
5. Ulož ako "[projekt]-kickoff.md"
6. Referuj počas celého projektu
```

# FÁZA 9: LAUNCH CHECKLIST

## 9.1 Pre-Launch (pred spustením)

```
CONTENT:
□ Všetok copy je finálny a schválený
□ Žiadne lorem ipsum
□ Kontaktné údaje sú správne
□ Právne texty (privacy policy, terms)
□ 404 stránka existuje

DESIGN:
□ Všetky breakpointy otestované
□ Hover/focus states fungujú
□ Dark mode (ak aplikovateľné)
□ Favicon a OG images

TECHNICAL:
□ SSL certifikát
□ Redirects nastavené
□ Analytics nainštalované
□ Forms otestované (odoslanie funguje)
□ Performance optimalizácia
□ SEO meta tags

TESTING:
□ Cross-browser testing
□ Mobile testing
□ Form testing
□ Link testing
□ Spell check
```

## 9.2 Launch Day

```
□ DNS propagácia dokončená
□ Analytics funguje
□ Forms fungujú
□ Backup vytvorený
□ Monitoring nastavený
□ Team notifikovaný
```

## 9.3 Post-Launch

```
□ Smoke testing v produkcii
□ Social media announcement
□ Email announcement
□ Google Search Console setup
□ Performance monitoring
□ Error tracking
□ Feedback collection
□ Prvé optimalizácie (po dátach)
```
